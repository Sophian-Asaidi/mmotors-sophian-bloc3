# Frontend M-Motors

Interface React + Vite pour les parcours client et administrateur.

## Lancement

```bash
cd mmotors/frontend
npm install
npm run dev
```

## Tests

```bash
cd mmotors/frontend
npm test
```

## Variables

Créer un fichier `.env` si besoin :

```bash
VITE_API_URL=http://localhost:8000
```

## Écrans
